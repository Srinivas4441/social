About This Application

What is it?

This web application is designed to analyze and predict social media usage time based on various activity metrics. By inputting data such as the number of posts per day, likes received per day, comments received per day, and messages sent per day, users can gain insights into their daily usage time on a selected social media platform.

What will the output display?

The output of the analysis will display:

  • A summary of the prediction results, including the mean squared error and RÂ² score, which indicate the accuracy of the prediction model.
  • A visual comparison of the actual vs. predicted daily usage time for the selected platform, displayed as a bar chart.

What technologies did we use?

We utilized a range of technologies to build this application, including:

  • Python: The backend logic for processing the data and performing the linear regression analysis.
  • Flask: The web framework used to build the web application and handle user interactions.
  • Pandas: For data manipulation and preparation.
  • Scikit-learn: For building and training the linear regression model.
  • Matplotlib & Seaborn: For creating the visualizations of the analysis results.
  • HTML/CSS: For structuring and styling the web pages, ensuring a user-friendly and visually appealing interface.
  • Bootstrap: For additional styling and responsive design features.
